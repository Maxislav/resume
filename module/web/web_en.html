Мониторинг

Для cлужбы спасения и помощи двухколесным участникам дорожного движения - МотоХелп. Сайт написан с учетом всевозможных пожеланий для более оперативного реагирование на инциденты в дороге. 80% функционала написан под требования МХ. Для "сторонних обывателей" основое преимущество - адаптация для смартфонов и работа приложения на малых скоростях передачи данных(GPRS) при большом количестве пользователей.

Принцип работы и конструкция: Сайт написан на платформе Leaflet. Парсер на JAVA слушает порт, принимает, разшифровывает и складирует принятые пакеты в базу данных MySQL. Страница с подключенным JavaScript с периодичностью опрашивает базу с "вопросом" появления новых координат. Все данные отображаются у всех пользователей входящих в группу. Сайт в непрерывно обновляется, временно сраничка автоматической регистрации отключена. Пробный вход demo demo.

Псевдо 3Д

Мотоцикл был смоделирован в 3dmax, отснят по кругу и зашит во flash.

Спросите кто его делал? Конечно же я :). Делалась в качестве учебы но с пользой для Украинского диллера по продажам этих мотоциклов.

Из дополнительных сведений - Основное время ушлона моделлинг, а именно 3 недели. Просчет одного цвета занимает от 4 до 8 часов. Программирования мало - но все равно крассиво. И сама детализация мотоцикла на порядок сложней моделинга авто из-за очень большого количества мелочевки (подножки винтики итп).
