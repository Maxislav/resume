For rescue and assistance Service moto/velo participant of traffic - МотоХелп. Site create according to the wishes MotoHelp Service for immediately response road accident. Most of the function site is notification MH. For the average user this functionality is hidden. The main advantage of the system, operating at slow communication.

The principle of operation and design: The site uses the library Leaflet. JAVA parser listens port, receives, decodes and stores the received packets into a MySQL database Page with javascript regularly polls the database analyzing the emergence of new coordinates. All data is displayed for all users in the group. Demo entrance: demo demo.
