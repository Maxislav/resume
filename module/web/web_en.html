AnyControl (Android)

Description.

The application is designed for a multiple devices and an unlimited number of commands. Any device controlled via SMS commands can be entered in the application memory. First, you create object that will monitor and on the page of all the commands add code associated with the object. The home page displays selected commands that you can send. On the page all the commands can be stored a full list of commands that you do not use, but can be useful. In order to command were available to send you need to mark them as favorites. Is also present the history of sent commands. If the command has been delivered her date marked in green color. In the application added map, which can be used to define you location, also to monitor the mobile devices that are attached to one of the servers monitoring. The application is available in English and Russian languages.

EnCon (Android)

Description.

Very simple application for memorizing words. just add the unfamiliar words and their translations and memorize when you have a free time. You can hide and show translations with a single touch. To fix the result, you need ten times correctly to make the word of mixed letters.
