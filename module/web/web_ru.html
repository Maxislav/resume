Для cлужбы спасения и помощи двухколесным участникам дорожного движения - МотоХелп. Сайт написан с учетом всевозможных пожеланий для более оперативного реагирование на инциденты в дороге. 80% функционала написан под требования МХ. Для "сторонних обывателей" основое преимущество - адаптация для смартфонов и работа приложения на малых скоростях передачи данных(GPRS) при большом количестве пользователей.

Страница с подключенным JavaScript с периодичностью опрашивает базу с "вопросом" появления новых координат. Все данные отображаются у всех пользователей входящих в группу.
